FullBright

A lightweight Fabric mod that lets you see in complete darkness with a single keypress.

How it works

Press B to toggle FullBright on or off. That's all there is to it — no configuration, no restart, no setup.

The default toggle key is B. You can change it any time from the Controls menu in Minecraft settings.

Features

  • True brightness — see in complete darkness without any light source
  • Completely silent — no particles or HUD icon
  • Lightweight — zero performance impact
  • Client-side only — works on any server, no install required server-side
  • Simple toggle — press B anytime to enable or disable
  • Changeable key — rebind the toggle key to your preference
  • Language support — available in English, Turkish, German, French, Spanish, Portuguese, Italian, Russian, and Hindi
